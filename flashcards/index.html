O que é Campo Harmônico?
O campo harmônico é um conjunto de acordes, geralmente tríades ou tétrades, construídos a partir das notas de uma escala musical específica

O que é Acorde?
Na música, um acorde é um grupo de duas ou mais notas tocadas simultaneamente. Essas notas são escolhidas de acordo com intervalos musicais, criando uma sonoridade específica que caracteriza o acorde. Os acordes podem ser maiores, menores, diminutos, aumentados, entre outros, dependendo dos intervalos entre suas notas.
